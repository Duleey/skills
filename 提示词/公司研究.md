1. 分析关键的问题
    1. 这家公司靠什么赚钱？客户是谁？为什么客户会持续付费？
        1. 需要包括收入结构（产品线/地区/客户类型/商业模式：订阅/交易/广告/硬件+服务），近3-5年变化与驱动。
        2. 单位经济模型（unit economics）：定价×毛利×获客成本×留存/复购×扩张；识别最核心的增长方程式。
    2. 有什么别人学不会的？（供给的独占性）
        1. 逐项判断（转换成本/网络效应/规模经济/品牌/数据/生态/监管/技术领先），找到证据的定量指标
        2. 护城河从哪里来？如何被量化验证？是否在变强或变弱？
        3. 对上下游有议价权吗？过去是否多次提价？提价后流失如何？
        4. 竞争对手能不能用不同路径绕过你？
        5. “Moat 的半衰期”：如果未来 5 年技术变了，护城河还在吗？
        6. 主要对手列表：明确差异化维度、价格/渠道/产品路线、指出最可能的“竞争拐点”
        7. 这个市场最终会不会变成低毛利的，如果是如何保持绝对的净利润值？
    3. 需求的稳定性？愿付费 + 频率 + 刚需程度 + 预算来源
    4. 增长靠直销/渠道/投放？渠道是否会反噬（佣金、议价）？
    5. 市场还有多少空间？
        1. 分别列出行业与市场空间：TAM/SAM/SOM + 增长驱动 + 渗透率逻辑
        2. 对“市场规模/份额”这类最容易编的数字：至少要求两家独立来源交叉验证。
        3. 明确“空间到底来自哪里”（替代旧方案/新增需求的渗透率提升/价格提升/品类扩张）。
    6. 还有哪些利润池没有释放
        1. 利润池在哪里：价值链哪一段拿走利润？（上游/中游/下游）
        2. 公司现在拿的是哪一段？未来能否“上移/下沉”吃到更多？
        3. 是否存在“交叉补贴”或“先亏后赚”的必经阶段？
2. 公司企业文化、组织管理方式是什么，适配商业模式吗？

    2. 组织效率如何？
    3. 它如何让战略更容易执行？如何让坏事更难发生？
    4. 文化和组织形式在规模化后会不会不可持续？
    5. 激励的奖金看收入还是自由现金流？会不会逼出“坏增长”？
3. 最主要的财务数据和用户数据是什么，最近三年的定量数据是什么，有什么规律需要参考
    1. 收入质量（一次性/递延/续费/客户集中度/合同期限）
    2. 毛利与费用结构（研发/销售/管理）与规模效应
    3. 净利润和自由现金流分别数据多少（偏差在哪里）
    4. ROE/ROIC 大概多少，ROIC 的来源是真竞争力还是杠杆/会计？
    5. 公司赚到的现金流是去回购注销股票、分红，还是盲目多元化扩张？管理层过往承诺兑现率；并购是补短板还是遮盖自身增长不足，并购后整合如何；回购是否在“对的价格”做；
    6. 股权稀释（SBC）是否侵蚀股东回报？
    7. 财务与会计质量
        * 收入确认是否复杂？一次性收入占比？是否提前确认？
        * 现金流 vs 利润的背离原因（递延收入、应收、存货）
