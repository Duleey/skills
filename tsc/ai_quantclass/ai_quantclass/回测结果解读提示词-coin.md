# 量化策略回测结果解读 - AI提示词

## 使用方法
将此提示词与策略回测仪表板截图一起发送给AI，即可获得专业的策略解读报告。

---

## 提示词

```
你是一位专业的量化策略分析师。请根据我提供的策略回测仪表板图片，进行全面深入的解读分析。

## 图表说明

仪表板通常包含以下图表（请根据实际图片内容分析）：

1. **标题栏**：显示核心指标（累计收益、年化收益、最大回撤、夏普比率）
2. **净值曲线**：对数坐标下的资金增长曲线
3. **回撤曲线**：历史回撤深度变化
4. **年度收益**：各年度收益率柱状图
5. **季度收益热力图**：按年份和季度展示收益
6. **月度收益**：月度收益率柱状图
7. **收益率分布**：收益率频率分布直方图
8. **杠杆比例**：策略杠杆使用情况
9. **多空比例**：多空仓位比例变化
10. **持仓品种数**：持仓分散程度

---

## 请按以下框架进行分析：

### 一、策略概况（30秒快速判断）

用一句话概括策略特征，给出整体评级（优秀/良好/一般/较差）。

### 二、收益能力分析

1. **绝对收益**
   - 累计收益率是多少？相当于本金翻了多少倍？
   - 年化收益率水平如何？（>100%极高，50-100%很高，20-50%较高，<20%一般）

2. **收益稳定性**
   - 各年度收益是否稳定？是否存在某一年异常高/低？
   - 月度胜率大约是多少？（正收益月份占比）
   - 是否存在明显的季节性规律？

3. **收益趋势**
   - 早期vs近期表现对比，是否存在Alpha衰减？
   - 净值曲线的增长是否平稳？

### 三、风险特征分析

1. **回撤分析**
   - 最大回撤是多少？发生在什么时期？
   - 回撤恢复速度如何？
   - 回撤超过10%、15%、20%的频率？

2. **波动特征**
   - 从收益分布图看，收益是否呈正态分布？
   - 是否存在"厚尾"特征（极端收益/亏损）？

3. **风险调整收益**
   - 夏普比率水平如何？（>3优秀，2-3良好，1-2一般，<1较差）
   - 收益回撤比（卡玛比率）如何？（>3优秀）

### 四、仓位管理分析

1. **杠杆使用**
   - 平均杠杆水平？最高杠杆？
   - 杠杆使用是否激进？

2. **多空偏好**
   - 策略偏多还是偏空？还是中性？
   - 多空比例波动是否剧烈？

3. **持仓分散度**
   - 平均持有多少个品种？
   - 持仓是否足够分散？

### 五、潜在风险提示

请识别以下风险信号（如存在）：

1. **过拟合风险**：早期收益是否异常高于后期？
2. **Alpha衰减**：近年收益是否明显下降？
3. **集中度风险**：持仓是否过于集中？
4. **杠杆风险**：杠杆是否过高？
5. **市场依赖**：是否只在特定市场环境下表现好？

### 六、综合评分

请给出100分制评分：

| 维度 | 分数 | 说明 |
|------|------|------|
| 收益能力 | /30 | |
| 风险控制 | /30 | |
| 稳定性 | /20 | |
| 可持续性 | /20 | |
| **总分** | **/100** | |

### 七、结论与建议

1. **一句话总结**：这是一个怎样的策略？
2. **适合人群**：适合什么风险偏好的投资者？
3. **配置建议**：建议配置比例？
4. **关注要点**：后续应重点监控什么指标？

---

## 输出格式要求

- 使用Markdown格式
- 关键数据加粗显示
- 风险提示用⚠️标注
- 优点用✅标注
- 结论简洁明了，避免过多专业术语
```

## 简化版提示词（快速解读）

```
请分析这张量化策略回测图，快速回答：

1. 策略赚了多少钱？（累计收益、年化收益）
2. 最大亏损是多少？（最大回撤）
3. 风险收益比如何？（夏普比率、收益回撤比）
4. 收益是否稳定？（年度波动、月度胜率）
5. 有什么风险隐患？（Alpha衰减、过拟合等）
6. 总体评价：推荐/谨慎/不推荐？为什么？
```

---

## 对比分析提示词（多策略对比）

```
我有多个策略的回测结果图，请帮我对比分析：

对比维度：
1. 收益能力：哪个策略收益最高？
2. 风险控制：哪个策略回撤最小？
3. 风险调整收益：哪个策略夏普比率最高？
4. 稳定性：哪个策略年度收益最稳定？
5. 综合排名：如果只能选一个，选哪个？为什么？

请用表格形式呈现对比结果。
```
