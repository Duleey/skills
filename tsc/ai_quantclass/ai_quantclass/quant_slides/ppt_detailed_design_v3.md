### 演讲稿
```
第一个坑：相信黑盒。

AI写的代码你不能直接用，必须自己能理解。如果有人问你"这个策略为什么会赚钱"，你答不上来，那就有问题。

第二个坑：过度拟合。

这是量化最经典的坑。你不断调整参数，让历史曲线越来越好看，但这些参数可能只是在拟合历史噪音，对未来没有预测能力。

怎么避免？用样本外测试，就是用一部分数据训练，另一部分数据验证。参数要简洁，太多参数一定有问题。

另外，如果你的策略夏普比率超过3，大概率有问题，要么数据错了，要么过拟合了。

第三个坑：忽视交易成本。

很多新手回测的时候假设零成本交易，实际上有手续费、滑点、冲击成本。高频策略尤其要注意这个。

第四个坑：幸存者偏差。

只测试现在还存在的股票，忽略了已经退市的。那些退市的很多是亏了钱的，你不把它们算进去，收益当然虚高。

记住：历史收益不等于未来收益，回测只是第一步。
```

## 推荐工具
- **Midjourney**: 适合艺术感强的封面和章节页，对中文提示词支持良好
- **DALL-E 3**: 适合图表和信息图，支持中文提示词
- **Stable Diffusion + ComfyUI**: 适合需要精确控制的技术图，建议使用英文提示词
- **可灵/即梦**: 国产AI绘图工具，中文支持最佳
- **Gamma.app / Beautiful.ai / 讯飞智文**: 可以直接生成整页PPT

## 提示词结构模板

```
【整体构图】
描述幻灯片的整体布局和目的，明确宽高比16:9，说明不要生成文字。

【背景设计】
描述背景的颜色、渐变、纹理、氛围等。

【主视觉元素】
详细描述每个视觉元素的位置、形态、颜色、风格。

【连接/装饰元素】
描述元素之间的连接关系、箭头、光效等。

【整体风格】
总结设计风格、色彩搭配、氛围感受。
```

## 提示词优化技巧

1. **明确不要文字**：每个提示词开头都说明"不要在图片中生成任何文字"，避免AI生成的文字出现拼写错误

2. **指定宽高比**：始终添加"宽高比16:9"确保PPT适配

3. **颜色一致性**：在所有提示词中使用相同的色板代码，如：
   - 深蓝背景：#0a192f、#1a365d
   - 强调色：#00d4ff（电蓝）、#ffd700（金色）
   - 成功/警告：绿色、红色/橙色

4. **分区描述**：按区域（左侧、右侧、顶部、底部）详细描述，AI更容易理解布局

5. **氛围描述**：加入"氛围"描述词，如"神秘"、"专业"、"活泼"、"科技感"

6. **参考风格**：可以提及参考风格，如"参考彭博终端风格"、"类似TradingView的美学"
